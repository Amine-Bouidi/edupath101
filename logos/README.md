# University logos

Drop PNG/SVG logo files here, named by the university's internal ID (see `UNIVERSITIES` in `index.html`).

Current expected files:

- `fpe.png` — Errachidia Multidisciplinary College (Faculté Polydisciplinaire)

If a file listed in `UNI_LOGOS` is missing, the university card falls back to the emoji defined on the university object (safe by design via the `onerror` handler in `renderUniGrid`).
